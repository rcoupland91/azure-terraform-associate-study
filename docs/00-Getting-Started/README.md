## Cheat sheet
- Terraform workflow: `write` ➜ `plan` ➜ `apply`.
- Core files: `main.tf`, `variables.tf`, `outputs.tf`.
- CLI help: `terraform <command> -help` for usage examples.

## Hands-on task
Run the following commands in an empty directory:
```bash
terraform init
terraform fmt
terraform version
```
